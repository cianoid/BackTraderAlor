# BackTraderAlor
Провайдер для автоторговли в [BackTrader](https://backtrader.com/) из [Alor Open API](https://alor.dev/docs). Использует библиотеку [AlorPy](https://github.com/cianoid/alor). Если новые бары получать по расписанию, а не по подписке, то потребуется библиотека [MarketPy](https://github.com/cia76/MarketPy).

### Для чего нужен
Чтобы торговые системы, написанные для BackTrader, можно было поставить на автоматическую торговлю с брокером Алор.
